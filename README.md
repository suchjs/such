# suchjs
An easy to use and extend mock tool.

## Documents
[中文文档](https://github.com/suchjs/such/wiki/%E4%B8%AD%E6%96%87%E6%96%87%E6%A1%A3)
## How to use
- node
```bash
npm install --save-dev suchjs
``` 
- browser
```html
<!--download the such.min.js-->
<script src="./dist/such.min.js"></script>
```
